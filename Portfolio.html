LIBRARY MANAGEMENT APP

GitHub

The Bookstore Library API is a RESTful API that provides endpoints for managing a collection of books. It allows the librarian adminstrative rights to perform operations such as creating, reading, and deleting books. The Bookstore library API implements authentication and authorization for secure access to protected routes. It ensures the confidentiality and integrity of sensitive user information using encryption, hashing, and salting techniques. It was developed using: React, CSS, Node.js, Express.js, SQLite.
